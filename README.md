## Larave + Breeze 
<img width="96" height="96" src="https://img.icons8.com/nolan/96/laravel.png" alt="laravel"/>

Laravel Breeze is a lightweight and simple starter kit that provides a basic authentication system for Laravel applications.

What does Breeze include?

Ready-made Login and Register pages

Password reset functionality

Email verification

Clean and minimal frontend (Blade, Vue, or React)

Simple and easy-to-understand structure

**If you want to understand the goal of this experience, you can check the schema in the “Idea” folder. Open the schema.drawio file, copy the code, and paste it into any online Draw.io editor to view it.**
